# Code Snippet Library

> Save, organize, and reuse code blocks from your Claude conversations. The snippet library gives you quick access to frequently used commands and code patterns.

---

## Saving a Snippet

There are two ways to save a code block to your snippet library:

### From a Code Block in Chat

1. When Claude responds with a code block, **long-press** (or right-click) the code block
2. Select **Save to Snippets** from the context menu
3. The snippet is saved with the code content and language tag

### From the Snippets Panel

1. In the chatroom, tap **Snippets** in the scrollable toolbar (alongside Memory, Skills, Jobs, Export)
2. Tap **+** to create a new snippet manually
3. Enter the code, give it a title, and select a language
4. Tap **Save**

---

## Accessing Snippets

1. In any chatroom, tap **Snippets** in the scrollable toolbar at the top of the chat
2. The Snippets panel opens, showing all your saved snippets
3. Tap any snippet to view its full content

---

## Using Snippets

From the Snippets panel, you can:

| Action | How |
|--------|-----|
| **Copy** | Tap the Copy button to copy the snippet to your clipboard |
| **Insert into chat** | Tap the snippet to insert it into your current message |
| **Delete** | Swipe left on a snippet and tap Delete |

---

## Tips

- **Save reusable commands:** When Claude gives you a useful one-liner (e.g., a complex `find` or `awk` command), save it as a snippet so you do not have to ask Claude again
- **Build a command reference:** Over time, your snippet library becomes a personalized command reference tailored to your specific workflows
- **Combine with inline execution:** After inserting a snippet into chat, you can ask Claude to modify it for your current use case, then use the Run button to execute it
- **Language tags:** Snippets preserve the language tag from the original code block, making it easy to identify shell commands vs. Python scripts vs. other languages
